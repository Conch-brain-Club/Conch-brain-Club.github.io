Welcome to ConchBrain DockerProxy

Dockerhub Proxy Service

Usage

Api: https://docker.conchbrain.club

Pull Image:
    docker pull docker.conchbrain.club/ubuntu
Global:
    sudo mkdir -p /etc/docker
    sudo tee /etc/docker/daemon.json <<-'EOF'
    {
        "registry-mirrors": ["https://docker.conchbrain.club"]
    }
    EOF
    sudo systemctl daemon-reload
    sudo systemctl restart docker
